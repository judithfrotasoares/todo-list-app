# 📋 To-Do List App

Um aplicativo simples de lista de tarefas feito com React.

## 🚀 Funcionalidades

- ✅ Adicionar tarefas.
- ✅ Marcar tarefas como concluídas.
- ✅ Deletar tarefas.
- 🎨 Estilização com CSS.

## 🛠️ Tecnologias Utilizadas

- ⚛️ React
- 🌐 JavaScript
- 📄 HTML/CSS
